🧰 Use Laravel-style Package in Plain PHP Project

📁 STEP 1: Folder Structure Setup

Create a folder like this:

simple-project/
├── index.php              ← Entry point of your PHP app
├── composer.json          ← Will contain dependency info
└── my-package/            ← Your custom package folder

📦 STEP 2: Initialize Composer Project

  1. Go to your project folder:
    cd simple-project
  2. Initialize Composer:
    composer init
  3. Edit composer.json and add:
{
  "name": "yourname/simple-project",
  "description": "Test project using a local package",
  "require": {
    "yourvendor/mypackage": "@dev"
  },
  "repositories": [
    {
      "type": "path",
      "url": "./my-package"
    }
  ],
  "minimum-stability": "dev",
  "prefer-stable": true
}

📥 STEP 3: Install Your Local Package

Run this command:

composer require yourvendor/mypackage:@dev

🧪 STEP 4: Use the Package in index.php

Create an index.php file with:

<?php
require __DIR__ . '/vendor/autoload.php';

use YourVendor\MyPackage\MyPackage;

$pkg = new MyPackage();
echo $pkg->hello(); // Output: Hello from MyPackage!

✅ STEP 5: Test It

Run this in terminal:

php index.php
OR: php -S localhost:8001

You should see:

Hello from MyPackage!

✅ DONE! You now have your Laravel-style package working in a plain PHP project.
